Inserir Links em seu Site

Links externos

links são utilizados para te direcionar a outra página e alavancar a estadia do usuário ao seu site
Para inserir o link na sua página, você deve inserir a tag:

<a href="Url do site/página do seu site"> nome do link</a>

para abrir o link em uma nova guia utilizar a tag "target="_blank" e rel="exteral"

Links internos

Para inserir links internos você precisa ter outra página criada no seu servidor.
Após isso é só inserir o caminho da sua segunda página na tag

<a href="caminho da próxima página do seu servidor" > nome do link</a>

  • rel="next": para página seguinte e
  • rel="prev": para página anterior

Links para download

Links para download são feitos usando a mesma tag anterior só que com o acréscimo DOS arquivos do seu servidor e utilizando as configurações:

  • download
  • type="application/formato"

Exemplo:

  • link para download em PDF
  • link para download em ZIP
  • link para download
