Carregando outras mídias no seu site

Audios

você consegue carregar outras mídias no seu site utilizando a tag "audio" com preload="metadata" Para pooder colocar outros formatos além do MP3 para rodar no site.

Para dar play no áudio vc vai utilizar as tags "controls" e "autoplay"

Exemplo:

<audio preload="metadata" controls autoplay>
<source src="../mídia/RQ-calcados/músicas/dance.mp3" type="audio/mp3">
</audio>

Observação

se quiseer que o audio fique em looping, é só adicionar a tag "loop" ao lado da tag "autoplay"

Exemplo:

<audio preload="metadata" controls autoplay loop>
<source src="../mídia/RQ-calcados/músicas/dance.mp3" type="audio/mp3">
</audio>
